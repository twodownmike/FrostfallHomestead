# Overworld Map Detail Sheet

Source: user-provided image from `/Users/crossmacserver/Downloads/Gemini_Generated_Image_ojc0knojc0knojc0.png`.

The sheet was copied to `source_sheet.png` and cropped into individual map-detail PNGs for snowy terrain, roads, resource nodes, and points of interest.

The provided image has an opaque grey sheet background, so the crops use background keying to remove the matte while preserving snowy highlights.

Small 48x48 terrain tiles prefixed with `tile_` were composed from the same sheet so the world grid no longer depends on the old Kenney terrain atlas.
